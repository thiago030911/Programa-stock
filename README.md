Sistema de Gestión de Stock
Este proyecto es una aplicación de consola desarrollada en C/C++ para gestionar el inventario de productos en una tienda o almacén. Fue creada utilizando el IDE Code::Blocks, y está diseñada para manejar de forma eficiente operaciones básicas de stock, como agregar, eliminar, buscar y actualizar productos.

Características Principales
📦 Gestión de Productos:

Agregar nuevos productos al inventario.
Actualizar la información de productos existentes (precio, cantidad, etc.).
Eliminar productos obsoletos o no disponibles.
🔍 Búsqueda Eficiente:

Buscar productos por nombre o código de identificación.
📊 Reportes de Inventario:

Generar listados del stock actual.
Mostrar productos con niveles bajos de existencias.
Tecnologías Utilizadas
Lenguaje de Programación: C/C++
Entorno de Desarrollo (IDE): Code::Blocks
Compilador: MinGW (incluido en Code::Blocks).
